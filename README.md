# Pocket-Budget
A java based expense tracker for managing personal finances and tracking expenses.

Features:
-Add income and expense transactions
-View the recorded transactions and net balance
-Delete transactions
-Simple Menu driven interface

Technologies used:
-Java
-Ecllipse IDE
-Object-oriented programming

Future improvements:
-File-based data persistence
-GUI using JavaFX or Swing
-Monthly reports and analytics
-User authentication
